WordPress plugin to manage and display albums of photos

== Description ==

WordPress plugin to display photos sorted into photo albums. Albums are generated automatically and then modified as needed.

== Features ==

*   Automatically read photos from a user-defined folder
*	Create multiple photo albums
*	Read file with meta data
*	Private by default

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/household-photos` directory, or install the plugin through 
the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Household Photos screen to configure the plugin.
